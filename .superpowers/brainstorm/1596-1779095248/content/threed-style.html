สไตล์การแสดงผล 3D ของบ่อน้ำ

เลือกลักษณะการ render 3D ที่ต้องการ

A — Three.js / React Three Fiber

3D จริงด้วย WebGL · หมุนกล้องได้ · มี lighting · สวยที่สุด · ใช้ @react-three/fiber · Bundle ใหญ่ขึ้น ~200KB

B — Isometric SVG (คำนวณเอง)

วาด SVG แบบ isometric projection · ไม่ต้องใช้ library เพิ่ม · เบา · แต่ไม่สามารถหมุนกล้องได้ · shape ที่ไม่ใช่สี่เหลี่ยมทำยาก

C — Three.js + Custom Extrusion

ใช้ Three.js ExtrudeGeometry จาก polygon ที่ user วาด · หมุนกล้องได้ · รองรับทุก shape · Recommended สำหรับโจทย์นี้
